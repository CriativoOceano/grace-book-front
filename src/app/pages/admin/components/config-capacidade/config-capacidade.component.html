Informações Importantes

Chalés Disponíveis

Este limite se aplica a reservas que incluem chalés. Quando atingido, novas reservas com chalés serão bloqueadas.

Antecedência Mínima

Os clientes só poderão fazer reservas com pelo menos este número de dias de antecedência.

Capacidade Máxima de Pessoas

Este limite define o número máximo de pessoas que podem ser incluídas em uma única reserva.
